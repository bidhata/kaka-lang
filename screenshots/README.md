# Screenshots

## Block Editor Screenshot

Place your block editor screenshot here as `block-editor.png`

The screenshot should show:
- Block palette with Bengali categories
- Workspace with colorful blocks
- Output console with results
- Action buttons

## Adding Screenshots

1. Save screenshot as `block-editor.png` in this directory
2. The README files will automatically display it

## Recommended Format

- Format: PNG
- Size: 1200px width recommended
- Quality: High for clarity
