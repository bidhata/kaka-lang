# Changelog

All notable changes to Kaka Lang will be documented in this file.

## [1.0.0] - 2024

### Added
- Initial release of Kaka Lang
- Core language features:
  - Variables with `কাকা রাখো`
  - Print statements with `কাকা বলো`
  - Input with `কাকা শোনো`
  - Conditionals with `যদি কাকা` and `নাহলে কাকা`
  - Loops with `যতক্ষণ কাকা`
  - Functions with `ফাংশন কাকা`
  - Arrays with bracket notation
  - Break and Continue statements
  - Logical operators (এবং, অথবা)
  - Boolean values (ঠিক, ভুল)
  - Null value (ফালতু)

- Web Interface:
  - Text editor mode with syntax highlighting
  - Virtual keyboard for Bengali keywords
  - Real-time code execution
  - Integrated output console

- Block Editor (Visual Programming):
  - Scratch-like drag-and-drop interface
  - 35+ block types across 10 categories
  - Color-coded blocks by category
  - Copy/paste and duplicate functionality
  - Code-to-blocks converter
  - Blocks-to-code generator
  - Block search and filter
  - Clear workspace feature
  - Nested block support (if-else, loops, functions)

- Documentation:
  - Complete language guide (English and Bengali)
  - README files (English and Bengali)
  - Block editor tutorial
  - Quick start guide
  - Deployment guide
  - Project structure documentation
  - Contributing guidelines

- Deployment:
  - Vercel configuration
  - Build scripts
  - Production-ready setup

### Features by Category

**Output & Input:**
- কাকা বলো (print)
- কাকা শোনো (input)

**Variables:**
- কাকা রাখো (declare)
- Variable assignment

**Control Flow:**
- যদি কাকা (if)
- যদি-নাহলে (if-else)
- যতক্ষণ কাকা (while loop)
- পুনরাবৃত্তি (repeat)
- ব্যাস কাকা (break)
- পরেরটা কাকা (continue)

**Math & Comparison:**
- Arithmetic operators (+, -, *, /, %)
- Comparison operators (>, <, >=, <=, ==, !=)

**Logic:**
- এবং (AND)
- অথবা (OR)
- নয় (NOT)

**Functions:**
- ফাংশন কাকা (function declaration)
- ফেরত নাও (return)
- Function calls

**Arrays:**
- Array creation with []
- Array access with index
- Array methods (push, pop, length)

**Other:**
- Comments with //
- Wait/delay blocks

### Safety Features
- 5-second execution timeout
- 1000-line output limit
- 1000 max loop iterations
- 100-line code paste limit
