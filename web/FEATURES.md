# Kaka Lang Web Interface Features

## Two Modes of Programming

### 1. Text Editor Mode
- Traditional code editor with syntax highlighting
- Virtual keyboard with Bengali keywords
- Real-time code execution
- Integrated output console
- Error reporting

### 2. Block Editor Mode (NEW!)
- Scratch-like visual programming interface
- Drag-and-drop block creation
- Color-coded blocks by category
- Nested block support for control structures
- Live code preview toggle
- Perfect for beginners

## Block Editor Features

### Visual Programming
- No typing required
- Intuitive drag-and-drop interface
- Visual feedback for block placement
- Color-coded categories

### Block Types
- Output: Print statements
- Variables: Declaration and assignment
- Control: If statements and while loops
- Functions: Definition and return statements

### Learning Tools
- Toggle between blocks and generated code
- See how visual blocks translate to text
- Immediate execution and feedback
- Clear error messages

### User Experience
- Smooth drag-and-drop
- Easy block deletion
- Nested container support
- Responsive layout
